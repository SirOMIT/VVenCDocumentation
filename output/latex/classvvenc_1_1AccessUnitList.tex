\doxysubsection{Detailed Description}
An Access\+Unit is a list of one or more N\+AL units, according to the working draft. All N\+AL units within the object belong to the same access unit.

N\+A\+L\+Units held in the Access\+Unit list are in E\+B\+SP format. Attempting to insert an \mbox{\hyperlink{structvvenc_1_1OutputNALUnit}{Output\+N\+A\+L\+Unit}} into the access unit will automatically cause the nalunit to have its headers written and anti-\/emulation performed.

The Access\+Unit owns all pointers stored within. Destroying the Access\+Unit will delete all contained objects.
