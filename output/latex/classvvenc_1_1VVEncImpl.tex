\doxysubsection{Detailed Description}
The class Hhi\+Vvc\+Dec provides the decoder user interface. The simplest way to use the decoder is to call \mbox{\hyperlink{classvvenc_1_1VVEncImpl_a809c348dba9e24a409c63298ad97221d}{init()}} to initialize an decoder instance with the the given V\+V\+C\+Decoder\+Parameters. After initialization the decoding of the video is performed by using the decoder() method to hand over compressed packets (bitstream chunks) in decoding order and retrieve uncompressed pictures. The decoding can be end by calling flush() that causes the decoder to finish decoding of all pending packets. Finally calling \mbox{\hyperlink{classvvenc_1_1VVEncImpl_a7e13b8ea1b81ac774cdabb8c1d68deef}{uninit()}} releases all allocated resources held by the decoder internally.
